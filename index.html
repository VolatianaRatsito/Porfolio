Framework Java Servlet

Développement d'un framework personnalisé pour applications web

À propos du projet

Ce projet consistait à développer un framework MVC personnalisé en Java utilisant les Servlets, afin de simplifier et d'accélérer le développement d'applications web. Le framework intègre un système de routage, l'injection de dépendances et un ORM léger.

Fonctionnalités principales

  • Système de routage avancé avec annotations
  • Injection de dépendances automatique
  • ORM léger pour la persistance des données
  • Système de templates pour les vues
  • Gestion des sessions et authentification
